# 📊 Retail Sales Data Analysis & Power BI Dashboard

## 📌 Project Overview
This project focuses on analyzing retail sales data to uncover business insights and present them through meaningful visualizations and an interactive Power BI dashboard. The goal of this analysis is to transform raw sales data into clear, actionable insights that help understand sales performance across different product categories, regions, and customer segments.

The project combines **data analysis using Python** with **business intelligence visualization using Power BI** to create a complete data analytics workflow.

---

## 🎯 Objectives
- Analyze retail sales data to identify patterns and trends
- Understand sales distribution across categories and regions
- Perform exploratory data analysis (EDA) using Python
- Build an interactive Power BI dashboard for business insights
- Present data in a clear and professional visual format

---

## 🛠 Tools & Technologies Used

### 🐍 Python (Data Analysis)
Python was used to perform **Exploratory Data Analysis (EDA)** and generate visual insights from the dataset.

Libraries used:
- **Pandas** – Data manipulation and analysis
- **Matplotlib** – Data visualization
- **Seaborn** – Statistical data visualization
- **Plotly** – Interactive and animated charts

### 📈 Power BI (Dashboard Development)
Power BI was used to design an **interactive business dashboard** that summarizes the key insights from the dataset in a visually appealing format.

---

## 📂 Dataset Description
The dataset contains information related to retail sales transactions including product details, customer segments, geographic locations, and sales values.

Key attributes used in the analysis include:

- **Sales** – Revenue generated from each transaction  
- **Category** – Product category classification  
- **Sub-Category** – Detailed product grouping  
- **Region** – Sales region information  
- **State** – Geographic location of sales  
- **Segment** – Customer segment classification  
- **Month & Year** – Time-based sales data for trend analysis  

These features help analyze how sales vary across **products, locations, and time periods**.

## 🔎 Exploratory Data Analysis (EDA)

The analysis process involved several steps:

### 📥 Data Loading
The dataset was imported and examined to understand its structure and available features.

### 🧹 Data Preparation
Basic data validation and preprocessing steps were performed to ensure the dataset was suitable for analysis.

### 📊 Data Visualization
Several visualizations were created to explore patterns in the data, including:

- Sales distribution across product categories
- Regional sales performance
- Monthly sales trends
- Top-performing products
- Customer segment analysis

Interactive visualizations were also created using **Plotly**, enabling dynamic exploration of the data.

---

## 📊 Power BI Dashboard

A professional **Retail Sales Analytics Dashboard** was created in Power BI to present the insights in an interactive format.

The dashboard includes the following visual components:

### 💰 Total Sales KPI
A card visualization highlighting the overall sales performance.

### 🛍 Sales by Category
A column chart displaying how sales are distributed across different product categories.

### 🌍 Sales by Region
A bar chart showing regional sales contributions.

### 📦 Sales by Sub-Category
A detailed breakdown of sales by product sub-categories.

### 📈 Monthly Sales Trend
A line chart that visualizes sales growth and trends over time.

### 🗺 Sales by State
A geographic map visualizing sales distribution across states.

### 👥 Sales by Customer Segment
A pie chart showing how different customer segments contribute to overall sales.

### 🎛 Interactive Filters
Slicers were added to allow users to filter the dashboard by:
- Region
- Category

This enables dynamic and interactive data exploration.

## 📌 Key Insights

Some key insights obtained from the analysis include:

- Sales performance varies significantly across product categories.
- Certain regions contribute more strongly to overall revenue.
- Some product sub-categories outperform others in terms of sales.
- Sales trends fluctuate across different months and years.
- Customer segments contribute differently to total revenue.

These insights help businesses better understand their sales performance and identify potential areas for improvement.
